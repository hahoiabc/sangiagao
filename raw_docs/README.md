# Thư mục Raw Docs

Đặt tất cả tài liệu thô từ khách hàng vào đây trước khi chạy skill `/bmad`.

## Các loại tài liệu được hỗ trợ

- Biên bản cuộc họp (Meeting notes)
- Tài liệu yêu cầu nghiệp vụ
- User stories thô
- Flowchart mô tả (dạng text)
- Email/chat trao đổi với khách hàng
- Tài liệu hiện tại của hệ thống cũ
- Bất kỳ tài liệu nào mô tả yêu cầu sản phẩm

## Định dạng được hỗ trợ

- `.md` - Markdown
- `.txt` - Plain text

## Ví dụ

Xem file `vi-du-yeu-cau-khach-hang.md` trong thư mục này để hiểu cách chuẩn bị tài liệu.
